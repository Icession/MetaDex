# MetaDex

**M**ost **E**ffective **T**actic **A**vailable — a Pokemon battle-analysis
assistant with an original electric-themed AI mascot. Reads your lineup and the
opponent's, gives mathematically-exact matchup / EV / move / item advice
deterministically, then narrates it in a lively in-character voice.

See `ROADMAP.md` for the plan and `CLAUDE.md` for the architecture + rules.

## Quick start
```bash
pnpm install
pnpm --filter @metadex/engine test   # should print 12 passing tests
```
